## What it does
A custom WordPress WooCommerce theme tailored for a premium tourism guide. Features an overhauled booking flow, strict validation, and deep SEO optimization.

## Tech Stack
WordPress · WooCommerce · PHP · ACF · Vanilla JS · Rank Math PRO · LiteSpeed Cache

## Key Engineering Decisions
- **Mobile-First Checkout UX:** Redesigned and overhauled the standard WooCommerce checkout flow into a native-app-style mobile experience to maximize conversion rates on mobile devices.
- **Strict Client-Side Validation:** Implemented robust Vanilla JS validation to prevent erroneous booking requests (dates, guest count) before they ever hit the server.
- **Automated SEO Infrastructure:** Configured automated Schema.org markup generation and dynamic WebP image conversion to maximize organic SEO performance and load speeds.
